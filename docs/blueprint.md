# **App Name**: DulceCaramelo Portfolio

## Core Features:

- Product Showcase: Display the different product categories of DulceCaramelo.
- About Us Section: Showcase the history and values of DulceCaramelo.
- Contact Form: Enable potential distributors to contact DulceCaramelo.
- Store Locator: Allows users to use their current location to locate DulceCaramelo distributors. If location fails, can enter the desired location via text

## Style Guidelines:

- Primary color: Soft rose (#F4B4C3), evoking sweetness and warmth.
- Background color: Very light rose (#F9E4E9), almost white, creating a gentle and inviting backdrop.
- Accent color: Pale yellow (#F5EAA7), an analogous color which evokes energy and happiness.
- Headline font: 'Playfair', a serif font lending an elegant feel; body text: 'PT Sans', a humanist sans-serif to make reading easy.
- Use custom-designed icons that mimic a logo to complement the sweet theme.
- Use high-quality images of candies to highlight the product showcase section.
- Subtle animations and transitions, such as candies floating on scroll, to enhance the user experience.
